延迟效应定量分析

1. 背景与概念

在网络同步的实时应用中，由于数据传输存在固有延迟，不同客户端对同一事件的感知会出现差异。本文旨在定量分析这种延迟对双方交互（如攻击判定）产生的影响。

1.1 感知的定义

对于任意实体或现象 σ\sigmaσ，在客户端 xxx 的感知下记作 σx^\sigma^{\hat{x}}σx^。

1.2 符号说明

符号 含义

Pxy^(t)P_x^{\hat{y}}(t)Pxy^​​(t) 在 yyy 的感知中，xxx 在时刻 ttt 的位置
Δαpx(t)\Delta_\alpha p_x(t)Δα​px​(t) 位置变化量：px(t)−px(t−α)p_x(t) - p_x(t - \alpha)px​(t)−px​(t−α)

2. 基本假设

为简化分析，作出以下关键假设：

  • 假设1：服务器内部处理延迟为零，所有延迟仅来源于网络传输。
  • 假设2：位置信息在客户端与服务器之间按时间顺序可靠传输。
  • 假设3：每个客户端对自身位置的感知是即时的，即 px(t)=Pxx^(t)p_x(t) = P_x^{\hat{x}}(t)px​(t)=Pxx^​(t)。

3. 延迟下的感知推导

3.1 关键参数定义

定义半程时延 rrr 和全时延 RRR：

r=τA+τB2,R=2r=τA+τB r = \frac{\tau_A + \tau_B}{2}, \quad R = 2r = \tau_A + \tau_B r=2τA​+τB​​,R=2r=τA​+τB​

3.2 感知位置的基本关系

客户端 A 发出的位置信息需要经过时间 rrr 才能被 B 完全接收。因此，双方对彼此位置的感知满足：

PAB^(t+r)=pA(t) P_A^{\hat{B}}(t + r) = p_A(t) PAB^​(t+r)=pA​(t) PBA^(t+r)=pB(t) P_B^{\hat{A}}(t + r) = p_B(t) PBA^​(t+r)=pB​(t)

3.3 感知距离的表达

在任意时刻 ttt，双方感知到的彼此距离为：

  • 在 B 的感知中：

    DB^(t)=PBB^(t)−PAB^(t)=pB(t)−pA(t−r) \begin{aligned} D^{\hat{B}}(t) &= P_B^{\hat{B}}(t) - P_A^{\hat{B}}(t) \\ &= p_B(t) - p_A(t - r) \end{aligned} DB^(t)​=PBB^​(t)−PAB^​(t)=pB​(t)−pA​(t−r)​

  • 在 A 的感知中：

    DA^(t)=PBA^(t)−PAA^(t)=pB(t−r)−pA(t) \begin{aligned} D^{\hat{A}}(t) &= P_B^{\hat{A}}(t) - P_A^{\hat{A}}(t) \\ &= p_B(t - r) - p_A(t) \end{aligned} DA^(t)​=PBA^​(t)−PAA^​(t)=pB​(t−r)−pA​(t)​

3.4 感知距离的关联

将以上两式结合，可得双方感知距离的关系：

DB^(t)−DA^(t)=[pB(t)−pA(t−r)]−[pB(t−r)−pA(t)]=[pB(t)−pB(t−r)]+[pA(t)−pA(t−r)]=ΔrpB(t)+ΔrpA(t) \begin{aligned} D^{\hat{B}}(t) - D^{\hat{A}}(t) &= [p_B(t) - p_A(t - r)] - [p_B(t - r) - p_A(t)] \\ &= [p_B(t) - p_B(t - r)] + [p_A(t) - p_A(t - r)] \\ &= \Delta_r p_B(t) + \Delta_r p_A(t) \end{aligned} DB^(t)−DA^(t)​=[pB​(t)−pA​(t−r)]−[pB​(t−r)−pA​(t)]=[pB​(t)−pB​(t−r)]+[pA​(t)−pA​(t−r)]=Δr​pB​(t)+Δr​pA​(t)​

即：

DB^(t)=DA^(t)+ΔrpB(t)+ΔrpA(t) D^{\hat{B}}(t) = D^{\hat{A}}(t) + \Delta_r p_B(t) + \Delta_r p_A(t) DB^(t)=DA^(t)+Δr​pB​(t)+Δr​pA​(t)

4. 攻击判定的临界条件

考虑 A 对 B 发动连续攻击（Combo）成功的条件。假设攻击有效距离为 3 个单位。

4.1 成功攻击的必要条件

  1. 条件一（A能打中B）：在A的感知中，存在某个时刻 t0t_0t0​ 使得距离小于攻击范围。

    DA^(t0)<3 D^{\hat{A}}(t_0) < 3 DA^(t0​)<3

  2. 条件二（B打不中A）：在B的感知中，任何时刻距离都大于攻击范围。由于A在t0t_0t0​发出的攻击指令需要经过时间rrr才能到达B，因此B在t0+rt_0 + rt0​+r时刻的感知距离必须大于3。

    DB^(t0+r)>3 D^{\hat{B}}(t_0 + r) > 3 DB^(t0​+r)>3

4.2 临界条件的推导

将t=t0+rt = t_0 + rt=t0​+r代入感知距离关联式：

DB^(t0+r)=DA^(t0+r)+ΔrpB(t0+r)+ΔrpA(t0+r) D^{\hat{B}}(t_0 + r) = D^{\hat{A}}(t_0 + r) + \Delta_r p_B(t_0 + r) + \Delta_r p_A(t_0 + r) DB^(t0​+r)=DA^(t0​+r)+Δr​pB​(t0​+r)+Δr​pA​(t0​+r)

我们需要将DA^(t0+r)D^{\hat{A}}(t_0 + r)DA^(t0​+r)用DA^(t0)D^{\hat{A}}(t_0)DA^(t0​)表示。注意到：

DA^(t0+r)=pB(t0)−pA(t0+r)DA^(t0)=pB(t0−r)−pA(t0) \begin{aligned} D^{\hat{A}}(t_0 + r) &= p_B(t_0) - p_A(t_0 + r) \\ D^{\hat{A}}(t_0) &= p_B(t_0 - r) - p_A(t_0) \end{aligned} DA^(t0​+r)DA^(t0​)​=pB​(t0​)−pA​(t0​+r)=pB​(t0​−r)−pA​(t0​)​

两式相减得：

DA^(t0+r)−DA^(t0)=[pB(t0)−pB(t0−r)]−[pA(t0+r)−pA(t0)]=ΔrpB(t0)−ΔrpA(t0+r) \begin{aligned} D^{\hat{A}}(t_0 + r) - D^{\hat{A}}(t_0) &= [p_B(t_0) - p_B(t_0 - r)] - [p_A(t_0 + r) - p_A(t_0)] \\ &= \Delta_r p_B(t_0) - \Delta_r p_A(t_0 + r) \end{aligned} DA^(t0​+r)−DA^(t0​)​=[pB​(t0​)−pB​(t0​−r)]−[pA​(t0​+r)−pA​(t0​)]=Δr​pB​(t0​)−Δr​pA​(t0​+r)​

代入条件二：

3<DB^(t0+r)=DA^(t0+r)+ΔrpB(t0+r)+ΔrpA(t0+r)=DA^(t0)+ΔrpB(t0)−ΔrpA(t0+r)+ΔrpB(t0+r)+ΔrpA(t0+r)=DA^(t0)+ΔrpB(t0)+ΔrpB(t0+r) \begin{aligned} 3 &< D^{\hat{B}}(t_0 + r) \\ &= D^{\hat{A}}(t_0 + r) + \Delta_r p_B(t_0 + r) + \Delta_r p_A(t_0 + r) \\ &= D^{\hat{A}}(t_0) + \Delta_r p_B(t_0) - \Delta_r p_A(t_0 + r) + \Delta_r p_B(t_0 + r) + \Delta_r p_A(t_0 + r) \\ &= D^{\hat{A}}(t_0) + \Delta_r p_B(t_0) + \Delta_r p_B(t_0 + r) \end{aligned} 3​<DB^(t0​+r)=DA^(t0​+r)+Δr​pB​(t0​+r)+Δr​pA​(t0​+r)=DA^(t0​)+Δr​pB​(t0​)−Δr​pA​(t0​+r)+Δr​pB​(t0​+r)+Δr​pA​(t0​+r)=DA^(t0​)+Δr​pB​(t0​)+Δr​pB​(t0​+r)​

4.3 简化形式

注意到:

ΔrpB(t0)+ΔrpB(t0+r)=pB(t0)−pB(t0−r)+pB(t0+r)−pB(t0)=pB(t0+r)−pB(t0−r)=ΔRpB(t0+r)=ΔRPBA^(t0+R) \begin{aligned} \Delta_r p_B(t_0) + \Delta_r p_B(t_0 + r) &= p_B(t_0) - p_B(t_0 - r) + p_B(t_0 + r) - p_B(t_0) \\ &= p_B(t_0 + r) - p_B(t_0 - r) \\ &= \Delta_R p_B(t_0 + r) \\ &= \Delta_R P_B^{\hat{A}}(t_0 + R) \end{aligned} Δr​pB​(t0​)+Δr​pB​(t0​+r)​=pB​(t0​)−pB​(t0​−r)+pB​(t0​+r)−pB​(t0​)=pB​(t0​+r)−pB​(t0​−r)=ΔR​pB​(t0​+r)=ΔR​PBA^​(t0​+R)​

因此条件二简化为：

3<DA^(t0)+ΔRPBA^(t0+R) 3 < D^{\hat{A}}(t_0) + \Delta_R P_B^{\hat{A}}(t_0 + R) 3<DA^(t0​)+ΔR​PBA^​(t0​+R)

结合条件一，得到A能够成功发动Combo的感知距离必须同时满足：

3−ΔRPBA^(t0+R)<DA^(t0)<3 3 - \Delta_R P_B^{\hat{A}}(t_0 + R) < D^{\hat{A}}(t_0) < 3 3−ΔR​PBA^​(t0​+R)<DA^(t0​)<3

5. 核心结论

在所述假设下，客户端A能够在延迟环境中对B成功发动连续攻击的充分必要条件为：在A的感知中，存在一个时刻t0t_0t0​，使得其感知到的双方距离DA^(t0)D^{\hat{A}}(t_0)DA^(t0​)满足以下不等式：

3−ΔRPBA^(t0+R)<DA^(t0)<3 \boxed{3 - \Delta_R P_B^{\hat{A}}(t_0 + R) < D^{\hat{A}}(t_0) < 3} 3−ΔR​PBA^​(t0​+R)<DA^(t0​)<3​

其中：

  • r=τA+τB2r = \dfrac{\tau_A + \tau_B}{2}r=2τA​+τB​​ 为半程时延
  • R=2r=τA+τBR = 2r = \tau_A + \tau_BR=2r=τA​+τB​ 为全程时延
  • ΔRPBA^(t0+R)=ΔRpB(t0+r)=pB(t0+r)−pB(t0−r)\Delta_R P_B^{\hat{A}}(t_0 + R) = \Delta_R p_B(t_0 + r) = p_B(t_0 + r) - p_B(t_0 - r)ΔR​PBA^​(t0​+R)=ΔR​pB​(t0​+r)=pB​(t0​+r)−pB​(t0​−r) 表示B位置在RRR时间内的变化量

此式定量揭示了延迟如何创造了一个”可攻击窗口”，该窗口的宽度由B在RRR时间内的位移决定。
