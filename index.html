简明 Hexo 使用教程

这是一个面向初学者的快速入门指南，包含安装、常用命令、调试与部署要点。

1. 前提

  • Git：用于部署到 GitHub Pages 或拉取主题。

7. 主题与插件

  • 下载并切换主题：将主题放在 themes/ 目录并在 _config.yml 中设置 theme: 主题名。
  • 安装插件：使用 npm install <包名> --save 并在 Hexo 或主题文档中按需配置。

8. 常见问题与排查

  • 页面生成为空：先运行 hexo clean，然后 hexo g，查看终端报错。
  • 主题样式缺失：确认已安装主题依赖并执行 npm install；检查主题的 source、_config.yml 配置。
  • 部署不上去：确认 repo 地址与 branch 配置正确，并具有推送权限。

9. 进阶建议

  • 使用 Git 管理源码，部署只放静态文件到 gh-pages 分支。
  • 在本地添加 CI（GitHub Actions）自动化 hexo generate + hexo deploy。

10. 参考资源

  • 官方文档：https://hexo.io
  • 中文指南与社区主题：搜索主题仓库（例如 Hexo 主题 Next）
